# Daily Code

Ce projet rassemble simplement les exercices que je fais chaque jour.

C’est mon espace d’entraînement personnel : j’y mets du code, des essais, des petits problèmes à résoudre, et différents exercices pour progresser régulièrement.

Le but de ce dépôt n’est pas d’être un projet fini ou structuré comme une application complète, mais plutôt de garder une trace de ma pratique quotidienne et de mon évolution.

## Contenu

On peut y trouver, selon les jours :
- des exercices simples ;
- des tests de code ;
- des essais en C, C++ ou Rust ;
- des fichiers temporaires liés à mon apprentissage.

## Remarque

Ce dépôt sert avant tout d’espace d’entraînement.
Le contenu peut donc être variable, incomplet, ou peu organisé selon les exercices du moment.
